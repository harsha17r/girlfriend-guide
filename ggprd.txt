# Product Requirements Document (PRD) - The Girlfriend Guide

## Project Overview

A fun, interactive digital notebook titled "The Girlfriend Guide: Pocket Edition - Vol. 1" containing 24 relationship tips presented as a floating book with page-turning animations, set against an animated sky background.

## Technical Stack & Dependencies

- **Turn.js** - Page turning framework
- **GSAP (GreenSock Animation Platform)** - Version 3.x
  - Core library for animations
  - ScrollTrigger plugin for scroll-based effects
  - Draggable plugin for enhanced interaction
- **Unicorn Studio** - WebGL animated background (embedded)
- **Custom fonts** - Upload .woff2/.woff files
- **Audio files** - Page turn sound effects (.wav/.mp3)
- **t** - Core structure

## Visual Design & Assets Needed

### Book Structure

- **Book format** - Hardcover format using Turn.js `.hard` class for front/back covers
- **Closed book state** - Using Turn.js framework with GSAP animations for floating effect
- **Open book pages** - HTML content with custom styling via Turn.js
- **Page dimensions** - 400px width × 600px height (standard Turn.js dimensions)
- **Book cover design** - Front and back covers will be PNG images with complex textures and blending modes that cannot be replicated with code
- **Page classes** - Utilize Turn.js classes:
  - `.hard` for covers
  - `.odd`/`.even` for different page styling
  - `.shadow` for realistic page shadows

## Development Rules & Guidelines

### Code Structure
- All code must include standardized comments and follow legible code.

### Font Handling
- **Font fallbacks**:
  - Implement graceful fallbacks for all custom fonts
  - Display notification at bottom of screen if custom fonts fail to load
  - Use lightweight system fonts as fallbacks (Arial for sans-serif, Georgia for serif)
  - Fallback notification should be subtle but visible (e.g., small gray text in footer)
  - Fallback fonts should maintain readability and not break layout

### GSAP Animations
- **Required GSAP plugins**:
  - GSAP core (TweenMax/TweenLite)
  - ScrollTrigger for scroll-based animations
  - Draggable for enhanced interaction
- **Animation specifications**:
  - Book floating animation: Gentle hover effect with slight rotation (±3°) and subtle up/down movement
  - Page turn enhancement: Subtle acceleration/deceleration curves for realistic movement
  - Initial book presentation: Fade-in and scale-up from 80% to 100%
  - Interactive elements: Micro-animations on hover/focus for buttons and controls
  - Performance optimization: Use GSAP's force3D for hardware acceleration
- **Animation timing**:
  - Floating animation: 3-4 second loop with ease-in-out
  - Page turn enhancement: 0.8-1.2 seconds based on user interaction speed
  - Initial presentation: 1.5 second intro animation
  - Hover effects: 0.3 seconds with elastic easing

### Image Handling
- **SVG Controls**:
  - Rotation controls (0-360 degrees)
  - Scale controls (percentage-based)
  - Position controls (x, y coordinates)
  - Color adjustment functions for SVG elements
  - Opacity controls (0-100%)
  - Blend mode options

- **PNG Controls**:
  - Sizing options with aspect ratio lock/unlock
  - Position controls with alignment helpers
  - Rotation with angle presets

### Responsiveness
- Clear breakpoints for desktop and tablet versions
- Automatic scaling of all elements based on viewport
- Maintain aspect ratios when resizing
